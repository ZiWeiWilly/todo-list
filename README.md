# todo-list
Common Todo List

---
## Environment
python3.11.2

in root folder ```/``` run:

```pip install -r requirement.txt```

---
## Frontend Development
in folder ```/todo``` run:

```yarn start``` or ```npm start```

---
## Frontend Build
in folder ```/todo``` run:

```yarn build``` or ```npm build```

---
## Runserver
in folder ```/todo``` run:

```python manage.py runserver```
